Quick Table Reply Editor

Paste a table from email, Sheets, Excel, CSV, TSV, or Markdown; make fast temporary edits; then copy a clean table back into your email reply.

Paste table data

Workflow

  1. Copy the table from the email or data source.
  2. Paste it into the box on the left.
  3. Click any cell to choose the current row and column.
  4. Move, add, or delete columns and rows.
  5. Use Copy rich table, then paste into your email reply.

Everything runs in this browser page. No data is sent anywhere.
